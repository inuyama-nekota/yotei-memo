# 予定メモ v2.1

予定を大きく、簡単に確認するためのスマホ向けアプリです。

## 主な機能

- 場所・日付・時間・メモだけの簡単入力
- 日付はカレンダー選択
- 時間は時と分を選択、分は5分単位
- 近い予定から自動で並び替え
- 3日以内はピンク、1週間以内は黄色、それより先は水色
- 予定時刻を過ぎた予定は最下部へ移動し、薄く表示
- 過去の予定は削除ボタン以外を操作できない表示
- 端末内へ自動保存
- オフライン利用
- iPhone・Androidのホーム画面へ追加可能

## Macで動作確認

1. ZIPを展開します。
2. `予定メモを起動.command` を開きます。
3. 初回の警告では、右クリックして「開く」を選びます。
4. ブラウザで `http://127.0.0.1:8771` が開きます。

終了時はターミナルで `Control + C` を押します。

## GitHub Pagesで公開

このフォルダの中身をGitHubリポジトリの `main` ブランチへ置きます。
同梱のGitHub Actionsが公開処理を行います。

リポジトリの Settings → Pages → Build and deployment で
Source を `GitHub Actions` にしてください。

## スマホへ追加

### iPhone

Safariで公開URLを開き、
共有ボタン → `ホーム画面に追加` → `追加` を押します。

### Android

Chromeで公開URLを開き、
メニュー → `アプリをインストール` または `ホーム画面に追加` を押します。

## データについて

予定は端末のブラウザ領域に保存されます。
ブラウザのサイトデータを削除した場合、予定も消えます。
現時点ではクラウド同期やバックアップ機能はありません。
